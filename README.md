# Parallax Website Clone

## Description
This project is a single-page parallax scrolling website inspired by the W3Schools Parallax Template.
The theme focuses on clean design and smooth scrolling using pure HTML and CSS.

## Features
- Semantic HTML5 structure
- Fixed navigation bar with anchor links
- Three parallax sections
- Image gallery
- Google Maps iframe
- Responsive design with media queries
- External Google Fonts & Font Awesome icons

## Technologies Used
- HTML5
- CSS3
